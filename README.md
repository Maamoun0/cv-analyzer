# CV Analyzer - Precision Career Optimization

**CV Analyzer** is a high-performance platform designed to help job seekers crack the ATS (Applicant Tracking System) code. Using a hybrid architecture of Next.js and FastAPI, it provides instant ATS scoring, detailed SWOT analysis, and context-aware professional rewrites.

---

## Key Features

- **Advanced Analysis**: Deep extraction and analysis of CV content.
- - **Dynamic ATS Scoring**: Real-time feedback on keywords, formatting, and relevance.
  - - **SWOT Analysis**: Clearly identified Strengths, Weaknesses, Opportunities, and Threats.
    - - **Job-Specific Matching**: Compare your CV against any job description using semantic embeddings.
      - - **Professional Rewrites**: Interactive "Accept/Reject" workflow to improve specific CV sections.
        - - **Premium UI/UX**: Dark-themed, high-performance interface with smooth animations and glassmorphism.
         
          - ---

          ## Tech Stack

          - **Frontend**: Next.js 14, Tailwind CSS, Framer Motion.
          - - **Backend**: FastAPI, Python, PostgreSQL.
            - - **Deployment**: Vercel & Railway.
